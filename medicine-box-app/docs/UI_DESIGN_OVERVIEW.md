## 智能药盒APP - UI设计展示

---

## 📐 设计规范总结

### 适老化设计核心原则

✓ **最小字体**: 18pt (所有可读文本强制要求)
✓ **最小触摸目标**: 48x48 dp (按钮、图标)
✓ **高对比度**: #FAFAFA背景 + #37474F文字（对比度 > 7:1）
✓ **大组件**: 按钮56dp，输入框56dp，Tab栏72dp
✓ **清晰层级**: 大标题32-48pt，正文18-24pt
✓ **圆角友好**: 12dp圆角减少视觉锐度
✓ **温暖配色**: 琥珀色 + 淡蓝 + 薄荷绿（柔和不刺眼）

## ✨ 设计亮点

### 适老化友好
✓ 所有文本最小18pt（屏幕阅读器友好）
✓ 按钮最小48x48dp（触摸友好）
✓ 高对比度配色（WCAG AA标准）
✓ 减少操作层级（最多2-3层）
✓ 大间距，避免误触

### 视觉风格
✓ 温暖琥珀色主调（愉悦不刺眼）
✓ 柔和粉彩辅助色（医疗健康感）
✓ 圆角设计（12dp，减少锐度）
✓ 大图标（32-64dp）
✓ 清晰的视觉层级

### 交互友好
✓ 大按钮（56dp）
✓ 大输入框（56dp）
✓ 图标按钮48dp
✓ Tab栏72dp
✓ 可见性切换（密码/确认密码）
✓ 一键操作（少步骤）
